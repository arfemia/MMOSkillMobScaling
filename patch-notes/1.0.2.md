Tune mob scaling entirely in game, and make every live change stick. Requires MMO Skill Tree 1.5.0+ and Ziggfreed's CommonLib 1.3.0+.

- **New: an in-game admin panel.** Run `/mobscaling ui` (admin only) to open a config screen with four tabs: Global (enable scaling, active preset, intensity, rarity chance, player scaling, difficulty caps, distance escalation), Zone HUD and Mob Inspector HUD (enable, on-screen position + offsets, and their options), and Worlds - a full editor for per-world overrides where you add, edit, and remove a world's own tuning without touching a file. Each world row shows whether it is a shipped default or your own override; blank fields inherit the global value. Changes apply immediately, and the enable toggle tells you when a restart is needed.
- **New: your live tweaks are saved.** `/mobscaling intensity`, `/mobscaling hud`, and `/mobscaling preset` now save automatically and survive a restart (before, they were live-only and reset when the server restarted). The panel and the commands use the same save path, so it does not matter which you use.
