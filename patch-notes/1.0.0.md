The Phase-1 skeleton for MMO Mob Scaling, a standalone open-world mob difficulty-scaling companion to the MMO Skill Tree mod. This build lands the foundation only: a zero-cost registration toggle and the config. The scaling systems land in later phases.

- **New: zero-cost registration gate.** The plugin loads its config in `setup()` and applies a registration gate: when the config is disabled the mod registers NO systems and returns, so it carries no per-tick cost at all. Enabled, it takes the register branch (empty for now; the scaling systems land in a later phase).
- **New: `MobScalingConfig`.** An override-based config at `mods/MmoMobScaling/mob-scaling.json` (only your customizations are stored), carrying the SIMPLE-preset starter values: rarity spawn chance + weights, per-zone difficulty overrides, party-join and late-arrival knobs, the open-world aggregation mode, and region size.
